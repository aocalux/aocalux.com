What is Java Library School?

Java Library School is a simple java library / some java classes witch make(s) some features of the java console easier to use.

Who is the target group of Java Library School?

It's mainly targeted at people who are learning java. But can be used by everyone.

What feature does it have?

• Generating a random integer within a defined range.
• Sekure input for all data types
• Writing a 1 and 2d array of all data types
